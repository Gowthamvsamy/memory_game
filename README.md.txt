# Memory Game

This is a simple Memory Game built using JavaScript, HTML, and CSS. The game involves flipping cards to find matching pairs within a grid. The timer keeps track of the time taken, and the number of flips is also recorded.

## Features

- **Responsive Design**: The game adjusts to different screen sizes using Tailwind CSS Flexbox and Grid.
- **Timer**: Keeps track of the time taken to complete the game.
- **Flip Counter**: Counts the number of card flips made by the player.
- **Reset Button**: Resets the game, timer, and flip count.
- **Winning Condition**: When all pairs are matched, a winning message is displayed.

## Usage

- **Back Button**: Click to return to the home page.
- **Reset Button**: Resets the game to its initial state.
- **Gameplay**:
  - Click on a card to flip it and reveal the image.
  - Try to find the matching pair by flipping another card.
  - If the cards match, they will stay revealed. If not, they will flip back over.
  - The game ends when all pairs are matched.

## Technologies Used

- **HTML**: Structure the game elements.
- **Tailwind CSS**: Style the game and ensure responsiveness.
- **JavaScript**: Handle game logic, including timer, flip counter, and matching logic.
